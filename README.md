## MateBook

An application inspired by the Facebook UI for posting messages and photos.

## Features

- Post messages
- Display photos

## Built with

- JavaScript
- React
- Material UI
- CSS

## Run Locally

- Clone the repository or download the project files.
- Go to respective directory
- npm install
- npm run start
